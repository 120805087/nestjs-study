# nestjs-study

## Installation

```
npm i -g @nestjs/cli
nest new project-name
```

## Alternatives

```
git clone https://github.com/nestjs/typescript-starter.git project
cd project
npm install
npm run start
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

Open your browser and navigate to http://localhost:3000/
